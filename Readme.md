# 📊 School Performance & Attendance Dashboard

## Overview

This project is an interactive School Performance & Attendance Dashboard developed in Microsoft Excel. The dashboard provides a comprehensive overview of student performance, attendance, fee collection, and demographic information through dynamic visualizations and interactive filters.

The goal of this project is to transform raw student data into meaningful insights that help track academic performance and support data-driven decision-making.

---
## Project Features

* Interactive KPI Cards

  * Total Students
  * Pass Percentage
  * Average Attendance
  * Average Subject Marks
  * Fees Collected

* Dynamic Slicers

  * Class
  * Section
  * Gender
  * Grade
  * Fee Status
  * Pass/Fail

* Data Visualizations

  * Students by Class
  * Gender Distribution
  * Average Marks by Subject
  * Attendance by Class
  * Grade Distribution
  * Fee Status Analysis

---

## Dataset Information

The dataset contains 2,000 student records with information such as:

* Student Details
* Class & Section
* Attendance Percentage
* Subject Marks
* Grades
* Fee Status
* Gender Information

The data was structured to simulate a real-world school management environment for dashboard reporting and analysis.

---

## Key Insights

* Monitor overall student performance and attendance trends.
* Identify grade distribution across the school.
* Compare average marks across subjects.
* Analyze fee payment status.
* Explore student demographics through interactive filtering.

## Conclusion

This project demonstrates how Microsoft Excel can be used to build a professional and interactive dashboard for educational data analysis. By combining Pivot Tables, Pivot Charts, KPI cards, and slicers, the dashboard transforms raw student data into meaningful insights that are easy to understand and analyze.

The dashboard enables users to track student attendance, academic performance, grade distribution, and fee collection efficiently. This project helped strengthen my skills in data cleaning, data visualization, dashboard design, and analytical reporting while showcasing the practical use of Excel for business and educational analytics.
